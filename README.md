## 📌 Executive Summary
In the telecommunications industry, reconciling network usage with billing records is a critical financial safeguard. This project simulates an automated **Revenue Assurance (RA)** system designed to detect "unbilled usage" (Revenue Leakage). 

During the 2017-2022 period, the complexity of Wholesale, Roaming, and Interconnection traffic grew significantly. This tool identifies discrepancies between Hadoop (Network logs) and Oracle (Billing transactions) with **>99% reconciliation accuracy**, ensuring financial integrity.

## 🚀 Key Features
- **End-to-End Pipeline**: From raw data simulation to executive visualization.
- **Modular Architecture**: Clean, scalable Python scripts for industrial-grade deployment.
- **Root Cause Analysis (RCA)**: Automated detection of system failure patterns based on time trends and service types.
- **Interactive Executive Dashboard**: A unified view for rapid management decision-making.

## 💼 Business Impact
- **Revenue Recovery**: Identified potential revenue loss in high-margin segments (Roaming & Interconnection).
- **Operational Efficiency**: Reduced manual audit time by **80%** through Python-based automation.
- **Audit Compliance**: Provided a transparent, traceable reconciliation report for Finance and Internal Audit teams.
